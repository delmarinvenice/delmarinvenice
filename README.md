### "Somos uma maneira de o cosmos conhecer a si mesmo." — Carl Sagan

Não encaro a engenharia de software como um processo mecânico de montagem. Para mim, escrever código é manipular uma matéria fluida e invisível; uma tentativa de moldar uma **consciência assistida** que se expande a partir da nossa percepção biológica. 

Os sistemas que crio buscam o equilíbrio dos cristais e a resiliência mutável da água: transparentes, performáticos e destituídos de qualquer complexidade de infraestrutura que aprisione a liberdade criativa.
